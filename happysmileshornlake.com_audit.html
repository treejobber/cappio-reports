Key Performance Issues

Largest Contentful Paint (LCP): 21.82 seconds on mobile
This is the single most important Core Web Vital. Google's threshold is 2.5 seconds. Your site takes nearly 9x longer. This alone can tank your mobile search rankings. The cause is likely unoptimized images, render-blocking resources, or slow server response on mobile devices.
First Contentful Paint (FCP): 9.91 seconds
Users see nothing for almost 10 seconds on mobile. Google's threshold is 1.8 seconds. Most users will abandon your site before it even starts to load.
Total Blocking Time (TBT): 486ms on mobile
JavaScript is blocking user interaction for nearly half a second. Google's threshold is 200ms. Audit your third-party scripts and defer non-critical JavaScript.
Desktop is acceptable — LCP 1.99s, FCP 1.24s
Your desktop performance passes Google's Core Web Vitals thresholds. The massive gap between mobile and desktop suggests the issue is mobile-specific: unresponsive images, lack of lazy loading, or JavaScript that's heavier on mobile network conditions.
